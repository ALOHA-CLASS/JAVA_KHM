
중첩클래스(이너클래스)
: 클래스 내부에 선언한 클래스
- 특정 클래스에 관계를 맺는 클래스를 정의할 때 사용
- 각각 아우터/이너 클래스의 멤버에 쉽게 접근할 수 있다

* 분류 : 멤버 클래스, 로컬 클래스

  -멤버 클래스 
  	* 인스턴스 멤버 클래스	
	  : A 객체를 생성해야한 사용할 수 있는 B 이너클래스

	* static 멤버 클래스
	  : A 클래스로 바로 접근할 수 있는 B 이너클래스

  - 로컬 클래스
	: 메소드 내에 클래스 정의,
	  메소드가 실행될 때만 사용할 수 있는 이너클래스
