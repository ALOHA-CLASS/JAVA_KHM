
 중첩 인터페이스
 : 클래스 내의 정의한 인터페이스
  - 인터페이스가 해당 클래스 의존적인 기능을 정의할 때,
  - 이너 인터페이스로 작성한다.
   ex. 이벤트 처리


 익명 객체
  : 이름이 없는 객체
    - 단독으로 생성할 수 없고,   
      클래스를 상속하거나 인터페이스를 구현해야 사용가능

    ▶ 익명 자식 객체
    -익명 자식 객체 생성하는 방법
    부모클래스 객체명 = new 부모클래스(  )  {
	// 자식클래스로 정의할 변수/메소드
     };

   
     ▶ 익명 구현 객체
     -익명 구현 객체 생성하는 방법
     인터페이스 객체명 = new 인터페이스( ) {
	
	// 구현클래스를 정의
	// 추상 메소드를 오버라이딩
 
      };
