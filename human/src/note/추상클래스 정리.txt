    ▶추상 클래스

      추상?  
      -여러 대상으로부터 공통된 특성을 뽑아내는 것

      추상 클래스
      - 구현될 여러 클래스가 가져야할 공통된 메소드의 원형을 정의한 클래스
      - 키워드 : abstract

      추상 메소드
      - 메소드의 body { } 블록이 없이 원형만 정의한 메소드
     
     "추상 클래스를 구현"
      : 추상 클래스를 상속받아, 자식클래스에서 추상메소드를 오버라이딩 하는 것
        * 구현 클래스는 반드시 추상클래스의 추상메소드를 오버라이딩 해야한다. 
          (추상메소드를 오버라이딩 하지 않으면 에러발생)


     사례)
