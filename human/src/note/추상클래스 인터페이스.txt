 추상 클래스
  설계자가 추상클래스를 작성하고,
  코더가 실제 구현 클래스의 코드를 완성하는 식으로,
  설계와 구현을 분리하여 작업할 수 있다.

  추상메소드 - 자식 클래스는 반드시 추상 메소드를 재정의해야한다.

  #익명 구현 객체
    - 클래스를 정의하지 않고도, 이름 없이 구현된 객체  
   -  익명 ?  : 이름이 없는 것

       - 문법 형식
       인터페이스 변수 = new 인터페이스( )  {
	// 추상메소드의 실체 메소드 선언
	메소드1(  ){  }
	메소드2(  ){  }
        };
